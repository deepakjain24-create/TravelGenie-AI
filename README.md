**AI Travel Planner – TravelGenie AI**

**WHAT ❓❓**

TravelGenie AI created by me (Deepak Jain) is a tool to plan the vacations or trips with just few clicks 🌞

**Overview 🌟**

TravelGenie AI is a web-based AI travel planner that generates a personalized travel itinerary based on user inputs such as destination, budget, travel duration, travel style, and interests. The system provides a day-by-day itinerary, budget breakdown, activities, and travel tips.

This project demonstrates the use of recommendation systems and AI logic in travel planning.

**Features 😊**

1) Destination-based itinerary generation
2) Budget breakdown calculation
3) Day-by-day travel plan
4) Interest-based activity recommendations
5) Travel tips and practical information
5) Responsive modern UI
6) Works without login
7) Supports multiple destinations (Tokyo, Paris, Bali, New York)
8) Generic itinerary for any destination
